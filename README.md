Installation</h2>

1. Clone the repository:
   ```bash
   git clone https://github.com/Manish-Chatterjee/Weather-App-3D.git
   ```
2. Navigate to the project directory:
   ```bash
   cd weather
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
